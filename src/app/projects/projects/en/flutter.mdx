## Overview

**Flutter** is a cross-platform mobile application project built as a hands-on exploration of the **Flutter** framework and **Dart** programming language. The app reimplements UI designs from previous committee projects — including the Gadjah Mada Festival and Blasout websites — adapted into native mobile screens.

Built during the *7 Days of Code: Mastering Flutter User-Interface Design* course by Build With Angga (2023), this project focused on translating existing web designs into Flutter widget trees, exploring layout primitives, custom theming, and responsive mobile design patterns.

## Tech Stack & Key Competencies

**Framework:** Flutter (Cross-platform Mobile Development)

**Language:** Dart

**Focus:** UI Component Building, Layout Adaptation from Web to Mobile, Theming
