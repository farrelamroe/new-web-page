## Overview

**Converter** is a temperature unit converter — my very first personal project after completing the foundational HTML, CSS, and JavaScript curriculum. The app converts between Celsius, Fahrenheit, Kelvin, and Reamur, providing instant results through a clean, minimal interface.

Although simple in scope, this project marked the beginning of my journey into web development by putting core concepts into practice: form handling, event listeners, and real-time DOM updates from user input.

## Tech Stack & Key Competencies

**Languages:** HTML5, CSS3, JavaScript (Vanilla)

**Concepts:** Form Handling, Real-time DOM Updates, Basic Unit Conversion Logic
