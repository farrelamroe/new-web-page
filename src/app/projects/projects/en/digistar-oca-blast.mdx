## Overview

**Digistar OCA Blast** is a web application developed as the final capstone project of the *Digistar Club Front End Developer* track by Telkom Indonesia. The platform is designed around **OCA Blast** — Telkom's omnichannel communication and automation platform — and implements a fully functional AI-powered chatbot interface.

The application enables users to interact with OCA Blast's chatbot features through a modern, responsive UI. Built with Next.js and Tailwind CSS, the project emphasizes clean component architecture, fast page performance, and seamless navigation between modules.

> **Note:** To access the chatbot page, navigate to the `/chatai` route.

## Tech Stack & Key Competencies

**Framework:** Next.js (App Router), TypeScript

**Styling:** Tailwind CSS, Responsive Design

**Features:** Chatbot UI, Route-based Navigation, API Integration
