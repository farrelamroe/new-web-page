## Overview

**Bookshelf Apps** is a browser-based book tracking application that allows users to record book titles, authors, publication years, and their reading status (read or unread). Built as both a personal side project and a course assignment in the front-end web development track, the app demonstrates core JavaScript DOM manipulation and local state management patterns.

Users can add new books, move books between "Read" and "Unread" shelves, and delete entries — all managed entirely in the browser using `localStorage` for data persistence between sessions.

## Tech Stack & Key Competencies

**Languages:** HTML5, CSS3, JavaScript (Vanilla)

**Concepts:** DOM Manipulation, localStorage Persistence, Event-driven UI
