## Overview

**Profile Web** is my personal bio and portfolio landing page — one of my very first web projects, built as both a creative exercise and a course assignment during the front-end web development learning track. The page introduces my background, skills, and links, serving as a simple but meaningful starting point for my web development journey.

Built from scratch using **HTML, CSS, and JavaScript**, the project helped solidify foundational browser rendering concepts, CSS box model, and DOM manipulation before moving on to frameworks.

## Tech Stack & Key Competencies

**Languages:** HTML5, CSS3, JavaScript (Vanilla)

**Skills Demonstrated:** Page Layout, DOM Manipulation, Personal Branding
