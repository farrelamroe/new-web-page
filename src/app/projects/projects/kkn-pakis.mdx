## Overview

**Website KKN Pakis 2024** is an official documentation website developed as part of the Pakis 2024 Community Service Program (*Kuliah Kerja Nyata* / KKN) — a mandatory community outreach program for Indonesian university students. The site serves as the digital record and public showcase for all work programs carried out during the KKN activity period.

The website covers programs across two villages — **Ketundan and Pogalan** — as well as cross-cluster programs from the science, social science, agro, and medical clusters. Built using **Next.js** and **Tailwind CSS**, the site is fully responsive and structured for easy content navigation by local community stakeholders.

## Tech Stack & Key Competencies

**Framework:** Next.js (SSR, Static Site Generation)

**Styling:** Tailwind CSS, Responsive Layout

**Content:** Multi-cluster program documentation, Village-level activity records
