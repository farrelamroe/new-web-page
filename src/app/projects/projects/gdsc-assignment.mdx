## Overview

**GDSC Assignment** is a pixel-accurate frontend clone of a Gojek page, built as the required selection assignment for joining Google Developer Student Club (GDSC) UGM in 2022. The project was completed solo in **under 4 days** and served as a demonstration of foundational frontend skills — layout accuracy, responsive styling, and clean HTML/CSS structure.

The clone replicates Gojek's visual design system, including navigation, hero sections, feature cards, and footer layout, all built from scratch without CSS frameworks.

## Tech Stack & Key Competencies

**Languages:** HTML5, CSS3, JavaScript (Vanilla)

**Skills Demonstrated:** Layout Replication, Responsive Design, Component Structuring
