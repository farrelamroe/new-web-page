## Overview

**Backend Engineer Final Project** is a Go-powered REST API developed as the culminating project of Alterra Academy's *"Backend Engineer With Golang"* Independent Study (MBKM) program. The system is designed to support digital agriculture, exposing a comprehensive set of API endpoints covering core farming operations.

Key features include plant watering schedule management, real-time weather forecast integration, fertilizer tracking, plant profile management, and community-based agriculture advisory services. The project demonstrates production-grade Go backend engineering patterns — clean architecture, middleware chaining, database migrations, and structured error handling.

## Tech Stack & Key Competencies

**Language:** Go (Golang)

**Database:** PostgreSQL, GORM ORM

**Architecture:** RESTful API, Clean Architecture, Middleware Pattern

**Tools:** JWT Authentication, Swagger API Docs, Git/GitHub
