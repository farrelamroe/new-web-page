## Overview

**OmahTI Assignment** is a design-slicing project developed as the entry requirement for joining OmahTI, UGM's Faculty of Computer Science's student developer organization, specifically for the 2022 Front-End division. The task was to faithfully implement a provided design mockup into a live, functional web page — completed solo in **under 4 days**.

The project focused on precision CSS layout work, responsive breakpoints, and clean semantic HTML structure — core skills for any frontend engineer working from design handoffs.

## Tech Stack & Key Competencies

**Languages:** HTML5, CSS3, JavaScript (Vanilla)

**Skills Demonstrated:** Design-to-Code (Slicing), Responsive Layout, Semantic HTML
