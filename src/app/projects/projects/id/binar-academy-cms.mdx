## Ringkasan

**Binar Academy CMS** adalah Sistem Manajemen Konten yang memberdayakan instruktur dan administrator kursus untuk membuat, mengelola, dan mempublikasikan konten pendidikan di seluruh platform Binar Academy. Dibangun bersama LMS selama magang Frontend Engineer 5 bulan, proyek ini dikembangkan dalam tim Agile beranggotakan 15 orang.

Frontend CMS dibangun menggunakan **Vue.js**, menawarkan arsitektur reaktif dan berbasis komponen yang ideal untuk antarmuka admin padat konten. Tema **SCSS** kustom dan utilitas **Bootstrap** memastikan UI yang bersih dan konsisten. Integrasi API RESTful menangani alur penerbitan konten, unggahan media, dan fitur manajemen pengguna.

Kontrol versi dan tinjauan kode dikelola melalui **GitLab**, dengan demo sprint dan retrospektif yang sering dilakukan untuk menjaga kecepatan pengerjaan.

## Tech Stack & Kompetensi Utama

**Framework:** Vue.js (Arsitektur Komponen Reaktif)

**Styling:** Bootstrap, SCSS (Tema Kustom)

**Integrasi API:** API RESTful, Alur Penerbitan Konten

**Kontrol Versi:** GitLab, Alur Kerja Sprint Agile
