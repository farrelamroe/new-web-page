## Overview

**Blasout 2023** is the official website for Blasout — a major cultural and entertainment event organized by Balairungklass UGM. The site serves as the main information hub for the event, introducing Blasout to **500+ prospective new students** within the first week of launch.

Developed in a 7-person team, the website was built using **Next.js** and **Tailwind CSS**, focusing on a visually engaging design and smooth navigation. The project prioritized full cross-device responsiveness and a reusable component library to allow the team to rapidly iterate and ship updates.

## Tech Stack & Key Competencies

**Framework:** Next.js (App Router, Static Generation)

**Styling:** Tailwind CSS, Responsive Design

**Version Control:** GitHub (7-person team)
