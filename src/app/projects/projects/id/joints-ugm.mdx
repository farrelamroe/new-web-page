## Overview

**Joints UGM** is a suite of high-traffic event websites developed for the **Jogja Information Technology Session (JOINTS) 2023**, a major tech competition organized by UGM's Faculty of Computer Science. Built within a 15-person engineering team, the project covers multiple sub-sites: Techtalk, Jointscamp, and the main JOINTS competition platform.

The main competition website features a fully integrated registration and authentication system that managed access and data for **500+ participants and administrators**. Specialized competition modules were built for Competitive Programming (CP) and Capture The Flag (CTF) events, each with custom-tailored interfaces. **Redux** was used to manage complex asynchronous API state and real-time competition data across the platform.

## Tech Stack & Key Competencies

**Framework:** Next.js (SSR, Dynamic Routing)

**Styling:** Tailwind CSS, Custom UI Components

**State Management:** Redux (Async API State, Real-time Data)

**Features:** Authentication System, Multi-event Competition Modules, RESTful API Integration

**Version Control:** GitHub (15-person team)
