## Overview

**Bookshelf API** is a RESTful backend service for managing a digital book collection, built using **Node.js** and the **Hapi.js** framework. Developed as a course assignment in the backend web development track, this project bridges the gap between the frontend Bookshelf Apps and a real server-side data layer.

The API supports full CRUD operations — creating, reading, updating, and deleting book records — with in-memory data storage. It demonstrates core backend concepts including HTTP method routing, request validation, and JSON response structuring.

## Tech Stack & Key Competencies

**Runtime:** Node.js

**Framework:** Hapi.js (REST API Framework)

**Language:** JavaScript (ES6+)

**Concepts:** RESTful API Design, CRUD, HTTP Routing, Request/Response Handling
