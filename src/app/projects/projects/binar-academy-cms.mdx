## Overview

**Binar Academy CMS** is a Content Management System that empowers instructors and course administrators to create, manage, and publish educational content across the Binar Academy platform. Built alongside the LMS during a 5-month Frontend Engineer internship, this project was developed in a 15-person Agile team.

The CMS frontend is built using **Vue.js**, offering a reactive and component-driven architecture ideal for a content-heavy admin interface. Custom **SCSS** theming and **Bootstrap** utilities ensured a clean, consistent UI. RESTful API integrations handled content publishing flows, media uploads, and user management features.

Source control and code reviews were managed through **GitLab**, with frequent sprint demos and retrospectives to maintain delivery pace.

## Tech Stack & Key Competencies

**Framework:** Vue.js (Reactive Component Architecture)

**Styling:** Bootstrap, SCSS (Custom Theming)

**API Integration:** RESTful APIs, Content Publishing Flows

**Version Control:** GitLab, Agile Sprint Workflow
